How to add a language pack?

  • 1. Open file /web_database_multilingual/views/db_manager.html
  • 2. 726 lines 
    
var languages = [{
"name": "English (US)",
"code": "en_US",
}, {
"name": "Chinese (Simplified) / 简体中文",
"code": "zh_CN",
}, {
"name": "Chinese (Traditional) / 正體字",
"code": "zh_TW",
},]
  • 3. According to the example, add a dictionary object at the end. For example:
    
var languages = [{
"name": "English (US)",
"code": "en_US",
}, {
"name": "Chinese (Simplified) / 简体中文",
"code": "zh_CN",
}, {
"name": "Chinese (Traditional) / 正體字",
"code": "zh_TW",
},{
"name": "Japanese / 日本語",
"code": "ja_JP",
},]
  • 4. Add a language pack file to the path /web_database_multilingual/static/src/i18n.
  • 5. The format is: prefixi18n_ + Area codeja_JP + file format.json，
    The full file name is:i18n_ja_JP.json
  • 6. Open file:i18n_en_US.json，Copy content to i18n_ja_JP.json
  • 7. The string after : in the translation file i18n_ja_JP.json.
  • 8. Please refer to the region code in odoo for the region code. Menu path:Settings > Translation > language.
